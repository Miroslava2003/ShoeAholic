# ShoeAholic

ASP.NET Core MVC web application for managing shoes.

## Technologies
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap

## Features
- User authentication (Identity)
- Product management (CRUD)
- Admin area
- Shopping cart
- Wishlist

## Purpose
University project / Internship portfolio
